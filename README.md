# Desktop Soundcloud App

## **Warning!** 
**This software is work-in-progress**

## Why?
- Because *Official*  Desktop soundcloud app sucks and is pretty buggy

## Features
- Liked songs
- Shuffle
- Native Media Controls

## Requirements
- Rust 1.68.2
- Node ^18.0.0


## Usage
```shell
$ npm i
$ npm run tauri dev
```

`Not tested on windows lol`
